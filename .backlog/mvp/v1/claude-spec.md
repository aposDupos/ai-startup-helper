# Синтезированная спецификация: StartupCopilot

**Версия:** 1.0  
**Дата:** 2026-02-28

---

## 1. Описание продукта

**StartupCopilot** — AI-платформа Сбера для помощи школьникам (14–18 лет) и студентам (18–25 лет) в создании стартапов. Платформа проводит молодого предпринимателя от идеи до питча с помощью персонального ИИ-наставника на базе **GigaChat**.

### Позиционирование
- Новый продукт в экосистеме Сбера (не надстройка над существующими акселераторами)
- AI-first подход: ИИ — ядро продукта, а не дополнение
- Целевой рынок: Россия
- Язык интерфейса: русский

### Видение развития
1. **MVP (к августу 2026):** AI-наставник + образование + геймификация
2. **v2:** Проектная работа для прошедших этап питча
3. **v3:** Работа с трекерами (менторами/кураторами)
4. **v4:** Полноценная акселераторная платформа

---

## 2. Целевая аудитория

### Сегмент A: Школьники (14–18 лет)
- Учащиеся 8–11 классов и ССУЗов
- Мотивация: интерес к бизнесу, карьерные перспективы, преференции при поступлении в вуз
- Уровень: начинающий, нужны простые объяснения и примеры из жизни
- Особенности: требуется согласие родителей, геймификация — ключевой retention-инструмент

### Сегмент B: Студенты (18–25 лет)
- Бакалавры, магистры, аспиранты
- Мотивация: гранты ПУТП (до 1 млн ₽), дипломная работа в формате стартапа, карьера
- Уровень: средний, есть базовые знания, нужна методология и инструменты
- Особенности: могут привлекать в команды, готовы к более глубокому контенту

### Оба сегмента равноценны для MVP.

## 3. Архитектурные ограничения

### 3.1 GigaChat (обязательно)
- LLM от Сбера — основная модель для AI-агента
- Необходимо исследовать:
  - API GigaChat: function calling (tool use), streaming, embeddings
  - Лимиты и тарифы API
  - Качество ответов для бизнес-тематики
  - Возможность RAG (embedding model, vector search)

### 3.2 Потенциальная интеграция с экосистемой Сбера
- **SberID** — авторизация
- **SberClass** — образовательный контент
- **SberStudent** — связка со стажировками и программами

### 3.3 RAG (Retrieval-Augmented Generation)
- Нужна база знаний стартап-фреймворков (Lean Startup, BMC, JTBD, CustDev)
- Требует: embedding-модель GigaChat (или альтернатива), vector store
- Вариант vector store: Supabase pgvector или отдельное решение

## 4. Core Features (MVP — к августу 2026)

### 4.1 Авторизация и Onboarding
- Регистрация через SberID / email / VK
- Выбор роли: школьник / студент
- Мини-опрос: опыт, интересы, цели, наличие идеи
- Персонализация интерфейса и контента AI по результатам опроса

### 4.2 AI Startup Copilot (ядро)

Персональный ИИ-агент на GigaChat в формате **микса**:

**Формат взаимодействия:**
- Чат-интерфейс (основной)
- Пошаговые мастера (структурированные этапы)
- Интерактивные карточки (quick actions, templates)

**Тон общения:**
- Поддерживающий, но честный
- Не отталкивает от платформы
- Не навязчивый
- Адаптируется к возрастному сегменту (проще для школьников)

**Этапы:**
1. **Поиск идеи** — сократический диалог, генерация идей по трендам, оценка ICE
2. **Валидация** — AI-симуляция CustDev, анализ конкурентов, гипотезы
3. **Бизнес-модель** — интерактивный BMC с AI-подсказками, юнит-экономика
4. **MVP** — рекомендации по no-code, user stories, roadmap
5. **Питч** — генерация питч-дека, тренажёр питча (AI = инвестор)

### 4.3 Startup Academy (образование)
- Микро-уроки (5–10 мин) по стартап-методологиям
- Адаптивный контент: школьникам проще, студентам глубже
- Кейсы реальных стартапов
- Квизы после уроков
- RAG-контент из промодерированной базы знаний

### 4.4 Gamification Engine
- XP за активность (уроки, этапы, чат)
- Уровни основателя (Dreamer → Explorer → Builder → Launcher → Founder)
- Бейджи за достижения
- Стрики (серии активных дней)
- Лидерборды (по школе/вузу, по городу, общий)
- Сезонные челленджи

### 4.5 Startup Dashboard
- Визуальный прогресс по стадиям
- Метрики проекта
- AI-рекомендации «что дальше»
- Экспорт артефактов (BMC, pitch deck)

### 4.6 Профиль пользователя
- Навыки, интересы, роль (Hustler / Hacker / Hipster)
- Портфолио проектов
- Достижения и уровень

- Менторские сессии (расписание, заметки)

- Оценка проектов кураторами

### v4: Акселераторная платформа
- Виртуальные Demo Days
- B2B подписки для школ/вузов
- Интеграция с ПУТП, грантовыми программами
- Мобильное приложение

## 6. Технический стек

| Компонент | Технология | Обоснование |
|-----------|-----------|-------------|
| Frontend | Next.js 15+ (App Router) | RSC, SSR, streaming |
| Стили | Tailwind CSS v4 | Быстрая разработка, токены |
| Backend | Supabase (PostgreSQL + Auth + Edge Functions + Realtime) | BaaS, RLS, масштабируемость |
| AI/LLM | **GigaChat API** | Требование Сбера |
| AI Framework | LangChain.js / LangGraph | Агентные воркфлоу, tool calling |
| Vector Store | Supabase pgvector | RAG для базы знаний |
| Хранилище | Supabase Storage | Файлы проектов, pitch decks |
| Аналитика | PostHog | Open-source, product analytics |
| Хостинг | Vercel / Сберовский облачный хостинг | Зависит от требований ИБ |

---

## 7. Нефункциональные требования

- **Производительность:** TTFB < 200ms, AI-ответ streaming < 2s до первого токена
- **Масштабируемость:** до 10 000 одновременных пользователей к запуску
- **Доступность:** 99.5% uptime
- **Безопасность:** 152-ФЗ (персональные данные), парентский контроль для несовершеннолетних
- **Локализация:** русский язык (интерфейс, AI-ответы, документация)
- **Адаптивность:** десктоп + мобильный веб
- **Accessibility:** WCAG 2.1 AA (базовый уровень)

## 8. Ограничения и риски

| Риск | Вероятность | Влияние | Митигация |
|------|-------------|---------|-----------|
| GigaChat API ограничения (нет function calling) | Средняя | Критическое | Обёртка для structured output, fallback |
| RAG с GigaChat embeddings — низкое качество | Средняя | Высокое | Альтернативная embedding модель |
| AI-галлюцинации в бизнес-советах | Высокая | Высокое | RAG, промодерированная база, disclaimers |
| Низкая retention школьников | Высокая | Среднее | Геймификация, стрики, push |
| Юридические ограничения (несовершеннолетние) | Средняя | Высокое | 152-ФЗ, согласие родителей |
| Дедлайн август 2026 — амбициозный scope | Средняя | Высокое | Phased delivery, приоритизация фич |
